## Scopes
| Ticket ID | Ticket Title | User Story |
|---|---|---|
| [JP-01](readme.md#user-register) | User register | As a User, i can register as company or candidate |
| [JP-02](readme.md#update-profil-company) | Update profil company | As a company, i can update my profil |
| [JP-03](readme.md#update-profil-candidate) | Update profil candidate | As a candidate, i can update my profil |
| [JP-04](readme.md#get-list-company) | Get list company | As a candidate, i can see list of company |
| [JP-05](readme.md#get-detail-company) | Get detail company | As a candidate, i can see detail of company |
| [JP-06](readme.md#review-company) | Review Company | As a candidate, i can review company with rating |
| [JP-07](readme.md#get-list-review-company) | Get List Review Company | As a candidate, i can get list of all reviews of company |
